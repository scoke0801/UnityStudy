유니티 - C#스크립트
- 유니티는 C#언어를 지원한다. 엔진의 코어는 C++로 작성

Start, Update
- 유니티의 가장 기본적인 함수이자 이벤트 함수.
- 이벤트 함수?
>>> 특정 조건을 만족할 떄 시스템(유니티 엔진)에서 자동으로 호출해주는 함수.

유니티 주요 이벤트 함수
Awake
- 스크립트가 실행될 때 한번만 호출되는 함수
- 주로 게임의 상태 값 또는, 변수의 초기화에 사용
- Start함수가 호출되게 전에 맨 먼저 호출
- 스크립트가 비활성화 되어있더라도 실행될
- 코루틴으로 실행 불가

Start
- Update 함수가 호출되기전에 한 번만 호출
- 스크립트가 활성화돼 있어야 실행된다.
- 다른 스크립트의 모든 Awake가 모두 다 실행된 이후에 실행된다.
- 코루틴으로 실행 가능.

Update
- 프레임마다 호출되는 함수로 주로 게임의 핵심 로직을 작성한다.
- 스크립트가 활성화돼 있어야 실행된다.
- 호출 간격이 불규칙적
- 화면의 렌더링 주기와 일치.98

LateUpdate
- 모든 Update함수가 호출되고 나서 한 번씩 호출된다.
- Update함수에서 전처리가 끝난 후 실행되야 하는 로직에 사용한다.
- 스크립트가 활성화돼 있어야 실행된다.

FixedUpdate
- 물리 엔진의 시뮬레이션 계산 주기에 맞춰 실핸된다. 기본값은 0.02초
- 호출되는 주기가 일정하다.

OnEnable
- 게임 오브젝트 또는 스크립트가 활성화됐을 떄 호출
- 주로 사용자 정의 이벤트 연결에 사용자
- 코루틴 사용 불가

OnDisable
- 게임 오브젝트 또는 스크립트가 비활성화 됐을 때 호출된다.
- 이벤트 연결을 종료할 떄 사용한다.
- 코루틴 사용 불가.

OnGUI
- 레거시 GUI 관련 함수를 사용할 때 사용한다.

로컬좌표계와 월드좌표계
- 유니티의 씬 뷰는 무한한 3차원 공간을 표현
- 해당 3차원 공간에서 바뀌지 않는 기준이 되는 좌표를 월드 좌표 또는 글로벌 좌표라 한다.
- 로컬 좌표는 씬 뷰에 존재하는 모든 게임오브젝트의 개별 고유 좌표를 지칭.

접근 제한자
public: 외부 클래스(스크립트)에서 접근 가능
private: 동일 클래스에서만 접근 가능(외부에서는 접근 불가능)
protected: private와 동일하게 외부에서는 접근이 불가능하고, 상속받은 파생클래스에서만 접근 가능.
internal: 같은 어셈블리에서만 접근 가능. 클래스의 경우 접근 제한자를 생락하면 internal이 기본값으로 설정됨.
