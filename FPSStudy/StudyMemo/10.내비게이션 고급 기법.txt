[ 동적 장애물 ]
- 동적으로 변경되는 장애물에 NavMeshObstacle을 활용.
- NavMeshObstacle의 Carve속성 - 실시간으로 내비메시를 갱신
	자연스러운 연산은 가능하지만 연산 부하가 크다.
	하위 옵션을통해 최적의 설정값을 찾는다.
	Move Threshold : 속성값의 거리만큼 이동했을 때 내비메시를 갱신.
	Time To Stationary : 동일 위치에서 일정 시간 동안 정지했을 때 내비메시를 갱신한다.
	Carve Only Stationary : 정지 상태에만 내비 메시를 갱신한다.
	
[ Off Mesh Link Generation ]
- 내비게이션은 기본적으로 3D 메시의 정보를 기반으로 내비메시를 생성
- 따라서 서로 분리된 메시를 추적할 수 없음.
- 다만 Off Mesh Link를 통해 서로 분리된 메시를 연결할 수 있다.
- static 옵션 중 Off Mesh Link Generation 옵션을 선택.

[ 사용자 정의 Off Mesh Link ]
- 자동 생성 Off Mesh Link는 DropHeight 높이 이내인 조건만 만족하면 무조건 생성.
	Drop Height: Off Mesh가 생성되는 최대 높이를 지정.
	Jump Distance: 같은 높이에서 자앵물을 뛰어넘을 수 있는 거리를 설정.
- 게임 내 시나오리상 특정 지점에서만 링크를 허용해야 하는 등 조건이 있다면, 자동 생성 Off Mesh Link를 사용하기 적합하지 않음.
- Off Mesh Link 컴포넌트를 추가.
	BiDirectional : 양방향성 여부 설정.
	Activated: 런타임 시 Off Mesh Link를 활성/비활성화 하는 옵션.
	Auto Update Positions : 런타임 시 링크의 위치가 변경될 때 이를 반영해 링크의 위치를 갱신하는 것.
	
[ 에이전트 컨트롤 ]
- 이동 또는 회전 처리를 직접 구현해야 하는 경우, updatePosition / updateRotate 속성을 활용한다.
- updatePosition : 위치를 자동으로 이동시키는 옵션
- updateRotation : 자동으로 회전시키는 옵션
- remainingDistance : 목적지까지 남은 거리
- velocity : 에이전트의 현재 속도.
- desiredVelocity : 장애물 회피를 고려한 이동 방향.
- pathPending : 목적지까지의 최단 거리 계산이 완료되었는 지 여부.
- isPathStale: 계산한 경로의 유효성 여부(동적 장애물, OffMeshLink)
