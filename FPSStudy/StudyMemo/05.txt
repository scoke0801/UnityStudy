[ 유니티 총기류 발사 방식 ]
Projectile
- 총알이 물리적으로발사되는 방식, 총알이 직접 날아가 적에게 타격을 입힌다.
- 총알을 시각적으로 표현
- 모델을 렌더링하기에 많은 양의 총알을 생성하면 게임의 속도가 저하 되는 단점
- 로우폴리 모델을 사용하거나, 평면 메시 모델에 텍스쳐를 입혀서 사용하거나
- 오브젝트 풀을 사용하여 관리

RayCasting
- 저격용 총의 발사 루틴을 구현하는 데 자주 활용
- 실제 총알이 발사되는 방식이 아니라, 레이저와 같은 눈에 보이지 않는 광선을 발사해 적 또는 물체를 검출하는 방식
- 총을 쏘면 총구 화염이 표시됨과 동시에 멀리 떨어져 있는 적이 피격되는 효과를 연출할 수 있음

RigidBody
- 물리학 용어의 강체를 의미. 충돌 감지 똔느 물리 시뮬레이션을 위한 중요한 컴포넌트.
- Mass : 질량. 상대적인 의미로 1KG, 1G등의 무게 단위가 아님. 편의상 1kg로 가정하고 작업하는 것이 일반적이라고 함
- Drag : 이동할 때 적용되는 마찰 계수( 저항 )
- Angular Drag : 회전할 떄 적용되는 마찰 계수( 저항 )
- Use Gravity : 중력 적용 여부.
- Is Kinematic : 체크하면 물리 시뮬레이션을 이용해 이동하지 않고, Transform 컴포넌트를 이용해 이동. 즉 이 옵션을 체크하면 물레 엔진의 영향을 받지않는다.
- Interpolate : 물리력을 이용한 움직임이 끊어지는 현상이 발생할 때 보간해준다.
--- Interpolate : 이전 프레임의 Transform에 맞게 보간해준다
--- Extrapolate : 다음 프레임의 Transform에 맞게 보간해준다.
- Collision Dectection : 세밀한 충돌을 검출하기 위한 옵션
- Freeze Position : x,y,z축에서 해당 축으로의 이동을 막는다.
- Freeze Rotation : x,y,z축에서 해당 축으로의 회전을 막는다.

AddForce : 월드 좌표 기준으로 힘이 가해짐
AddRelatvieForce : 로컬 좌표 기준ㅇ으로 힘이 가해짐.

물리엔진 속성 설정 - Physiscs Manager
- Edit - Project Settings - Physics
- Default Material : 두 물체가 충돌했을 때, 물리적인 반작용에 대한 속성을 전역적으로 설정한다. None으로 설정하면 각 Rigidbody에서 개별적으로 물리적인 반작용을 설정할 수 있다.
- Bound Threshold: 충돌한 두 물체의 상대 속도가 이 값보다 자으면 충돌이 발생하지 않는다. 떨림 현상(Jitter)을 감소시키는 역할
- Layer Collision Matrix : 레이어 간의 충돌 감지 여부를 활성화 or 비활성화.

Collider
- Box / Sphere / Capsule 
- Mesh : CPU 부하가 가장 높은 Collider. 게임 월드가 복잡한 경우 or 미로같은 경우. mesh renderer는 제거하고 mesh collider로 확인 하는 경우.
- Wheel : 차량의 바퀴에 사용할 목적으로 제공. 바퀴의 서스펜션, 바닥과의 마찰계수, 미끄러지는 저항을 세밀하게 설정.
- Terrain : 유니티에 내장된 Terrain Engine을 이용해 생성한 지형에 적용되는 Collider, 지형 복잡도에 따라 부하 높아짐. 로우폴리 메쉬를 이용해 mesh Collider로 교체 가능.
- 연산속도 : Sphere > capsule > box

충돌 감지 조건.
- 충돌을 일으키는 양쪽 오브젝트에는 모두 Collider 컴포넌트가 존재.
- 움직이는 쪽 오브젝트에는 반드시 Rigidbody컴포넌트가 존재.

충돌 이벤트( 콜백 )
- 충돌이 발생하면 이벤트 함수를 호출
- 언제 충돌이 발생할 지 모르는 상태에서 충돌이 감지되는 시점에 시스템에 의해 호출되기 떄문에 이벤트라 함( 또는 콜백 )

Tag 활용
- 충돌이 발생했을 때, 식별하기 위한 값.

기즈모 활용
- Transform 컴포넌트만 가진 게임오브젝트는 씬 뷰에서 마우스로 클리갛거나 영역을 드래그해서 선택할 수 없음
- 해당 불편함을 해소하기 위한 방법으로 사용자 정의 기즈모를 추가
- 유니티 에디터에만 보이고 실행시에는 보이지 않는다.
